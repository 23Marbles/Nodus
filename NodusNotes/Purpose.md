# Purpose
Maths is beautiful, yet this beauty can be hard to find. Take, for example, π. Read as a number, this is simply an annoyingly long decimal that we can't fully write. This on of itself is not beautiful (unless you think all of maths is inherently beautiful). The real beaty comes when we can see it. When we see how it relates to e, i, radians, circles and the like, then, it is beautiful. This project seeks to bridge that gap, to transform from simple numeric analysis to something we can see and appreciate, to draw patterns from data and to appreciate mathematics in all it's beaty.
